WebRTC SFU 试点（Layer B：信令 + mediasoup-client）

本页通过 WebSocket 与 mediasoup 完成 摄像头 → SFU → 另一浏览器 Tab 的最小闭环。 不包含 Redroid / 安卓采集（Layer C）。需先执行 npm install 与 npm run build:client 生成 public/mediasoup-client.esm.js。 局域网访问时请在服务端设置 MEDIASOUP_ANNOUNCED_IP（填浏览器连 SFU 媒体面用的公网 EIP 数字，与 HTTPS 域名无关）。他人「仅观看」黑屏时：云安全组须放行 SFU 的 UDP 40000–49999；对方网络若限制 UDP 可能需 TURN（本 PoC 未内置）。刷新后看日志里的 bytesReceived 与 unmute 提示。
